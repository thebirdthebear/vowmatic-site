Help & Support

Everything you need to get started, troubleshoot, and get the most out of Vowmatic.

Install Vowmatic

Download Vowmatic (includes everything you need).

  1. Open the downloaded Vowmatic.dmg
  2. Drag Vowmatic to your Applications folder
  3. Double-click Vowmatic to launch — your first wedding is free, no account needed

Note: On first launch, macOS may show a "developer cannot be verified" warning. Right-click the app and choose "Open" to bypass this. This only happens once.

Run Your First Wedding

  1. Launch Vowmatic from your Applications folder
  2. A browser window opens. Click Choose Folder and pick the folder where your wedding footage lives
  3. Choose your editing software (Final Cut Pro, DaVinci Resolve, or Premiere Pro) and select Full Analysis
  4. Click Start — Vowmatic scans your footage and sorts every clip by event (~10 minutes)
  5. A review screen opens. Quickly check that clips are sorted correctly, then tell it which audio files go with which events
  6. Click Continue — Vowmatic syncs your cameras, builds multicam edits, and organizes everything. Go grab coffee — this takes about 30 minutes.
  7. When it's done, open the files it created in your editing software. Everything is ready to go.

Your Editor

Select your NLE below for a walkthrough video and import instructions specific to your editor.

Walkthrough

What you'll get

Vowmatic creates an FCPXML bundle that opens directly in Final Cut Pro:

Selects library

All your clips sorted into keyword collections by event. Favorites are marked. Shaky or unusable clips are flagged.

Multicam edits

A multicam timeline for each event. All cameras synced, cut decisions made. Use the Angle Viewer to switch angles.

Story edit

An AI-assembled highlight reel — your starting point for the final film.

Importing

  1. Open Final Cut Pro
  2. Go to File → Import → XML and select the .fcpxml file from your project folder
  3. FCP will create a new event with all selects, multicam timelines, and story edits organized inside
  4. Open a multicam timeline and use Window → Show in Workspace → Angle Viewer to switch cameras

Walkthrough

What you'll get

Vowmatic creates three XML files. Import them via File → Import:

premiere_selects.xml

All clips organized into bins by event with favorites marked. Import this first.

premiere_multitrack.xml

A sequence for each event with the autocut on top and synced cameras below. Blade through the autocut to reveal the camera you want.

premiere_story.xml

The AI-assembled story edit — same multi-track layout.

Importing

  1. Open Premiere Pro and create or open your project
  2. File → Import → select premiere_selects.xml first
  3. File → Import → select premiere_multitrack.xml
  4. File → Import → select premiere_story.xml for the story edit
  5. If prompted to relink media, click Locate and check Relink others automatically

Tips

Audio: Board audio is active on all sequences. Lav mics are imported but muted — unmute if you prefer lav audio.

Multicam switching: Select the synced camera clips, right-click, and choose Create Multi-Camera Source Sequence.

Troubleshooting

Media shows as offline: Go to File → Link Media, click Locate on any clip, navigate to your drive, and check "Relink others automatically."

Autocut track shows wrong angle: Make sure you imported premiere_multitrack.xml (not the selects file).

Walkthrough

What you'll get

Vowmatic creates FCPXML files optimized for DaVinci Resolve:

Selects library

All clips sorted into bins by event. Favorites marked. Shaky clips flagged.

Multicam edits

A multicam timeline for each event with synced cameras and Vowmatic's suggested cuts.

Story edit

An AI-assembled highlight reel — your starting point for the final film.

Importing

  1. Open DaVinci Resolve and create or open a project
  2. Go to File → Import → Timeline
  3. Select the .fcpxml file from your project folder
  4. Resolve will import the selects, multicam timelines, and story edit
  5. If media appears offline, right-click a clip and choose Relink Selected Clips

Tips

Timeline settings: Make sure your project timeline framerate matches your footage before importing.

Multicam: Switch angles directly in the Edit page timeline, or use Resolve's multicam viewer.

Upgrading After Your Free Wedding

After your first wedding, visit pricing to purchase credits or a subscription. We'll email you a personal license key. To activate it, open Terminal on your Mac and paste the command from your email. It looks like this:

python3 -c "import json; from pathlib import Path; p=Path.home()/'.wedding_analyzer/config.json'; d=json.loads(p.read_text()); d['api_token']='YOUR_KEY_HERE'; p.write_text(json.dumps(d,indent=2)); print('Done')"

Your email will have this command with your key already filled in — just copy, paste, done.

Common Questions

What NLEs are supported?

Final Cut Pro, DaVinci Resolve, and Premiere Pro. Each gets native-format output optimized for that editor.

What cameras work?

Anything that shoots MP4 or MOV. Tested with Sony (A7, FX, A1), Canon (R5, R6), DJI drones, and GoPro.

Does my footage get uploaded?

No. All video and audio stays on your Mac. Only small frame thumbnails are sent to AI for classification.

How long does processing take?

About 45 minutes total, mostly unattended. The initial scan needs a brief review.

Can I change the edit after?

Absolutely. Everything opens as a normal timeline. Move cuts, swap angles, trim — it's a starting point.

What Mac do I need?

macOS 12+ on any Mac. Apple Silicon (M1+) is faster. Intel Macs work too.

Do credits expire?

Never. Buy a credit today, use it whenever you need it.

How do I update Vowmatic?

Download the latest DMG from our download page and replace the app in Applications.

Troubleshooting

Vowmatic won't launch

Make sure you're on macOS 12+. If macOS blocks the app, right-click and choose Open (this only happens once). If issues persist, try re-downloading from our download page.

Clips not showing up in your editor

Make sure your footage drive is plugged in before opening the file. It links to your original footage — if the drive isn't connected, your editor won't find the clips.

Getting an error during scanning

This usually means your internet connection dropped. Vowmatic needs internet for the clip-sorting step. Check your connection and try again.

Some clips got sorted into the wrong category

That happens sometimes — AI isn't perfect. Use the Clip Review screen to drag misidentified clips to the right category.
